# Free Growth Plan — Desert Comfort Quotes

Goal: get first Phoenix HVAC leads and contractor buyers without paid ads.

## 1) Free hosting launch

Best no-cost options for this static MVP:

1. GitHub Pages
   - Cost: free
   - Good for: static HTML, SEO indexing, public demo link
   - Needs: GitHub token/account auth
2. Netlify Drop
   - Cost: free
   - Good for: fastest manual drag-and-drop deploy
   - Needs: browser login
3. Cloudflare Pages
   - Cost: free
   - Good for: fast global hosting + later custom domain
   - Needs: GitHub connection or direct upload

Recommended: GitHub Pages now, Cloudflare Pages later when custom domain is ready.

## 5) Contractor acquisition without ads

Offer:
- First 3 qualified leads free
- Exclusive lead only
- Pay only after they confirm lead quality

Message angle:
- “I’m launching a Phoenix AC quote-matching site and looking for 2–3 HVAC partners. I’ll send the first few qualified requests free so you can test quality.”

Daily free outreach:
- 10 emails/contact forms
- 5 phone calls
- 5 LinkedIn/Facebook messages

Qualification questions for contractors:
- Which ZIP codes do you want?
- Do you prefer repair, replacement, heat pump, maintenance?
- Can you handle same-day emergency calls?
- What is a qualified lead worth if exclusive?
