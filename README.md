# Desert Comfort Quotes — Phoenix HVAC Lead Gen MVP

Static MVP landing page for Phoenix AC repair, AC replacement, and heat pump quote matching.

## Files
- `index.html` — consumer landing page + multi-step quote form (demo/localStorage)
- `privacy.html` — basic privacy policy
- `terms.html` — basic terms/disclaimer
- `robots.txt`, `sitemap.xml` — basic SEO indexing files

## Current MVP limitation
The form currently stores submissions in browser localStorage only. Production should connect to Airtable, Google Sheets, Zapier, Supabase, email, or CRM webhook.
